## Deskripsi
PINKY PONG EXTREME adalah permainan Pong berbasis Computer Vision
yang dikendalikan menggunakan gesture tangan secara real-time
(open hand dan fist) melalui webcam dengan OpenCV.

## Teknologi
- Python
- OpenCV
- NumPy

## Cara Menjalankan
1. Install dependensi:
   pip install opencv-python numpy
2. Jalankan file:
   Jalankan notebook `TinsariRauhana_2308107010038_UasViskom.ipynb`
3. Gunakan gesture:
   - Open Hand → paddle naik
   - Fist → paddle turun

## Kontrol Keyboard
- 1 : Camera Mode
- 2 : Arena Mode
- T : Ganti Tema
- R : Reset Game
- Q : Quit
